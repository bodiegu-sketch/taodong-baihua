原著：周金涛  |  白话翻译：Claude Code
周金涛（1972–2016），中信建投证券首席经济学家，被誉为"周期天王"。 他用康德拉季耶夫周期理论（康波）精准预测了多次经济拐点，留下"人生发财靠康波"的著名论断。 本白话版将原文逐段翻译为通俗易懂的文字，让每个人都能读懂经济周期。
